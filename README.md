**Developed by: Jaideep Singh**

## 🚀 Project Overview

SOLACE is an integrated, eco-friendly transportation management system designed to improve campus mobility. The platform combines **e-rickshaw booking** and **e-cycle rental management** into a single mobile application.

Built using **Android (Java & XML)** with **Firebase** as the backend, SOLACE provides:

- Real-time ride booking
- Smart lock integration for e-cycles
- Secure OTP-based ride verification
- Time-based fare calculation
- IoT-enabled hardware communication
- Secure authentication and payments

The goal of SOLACE is to create a **sustainable, efficient, and secure commuting ecosystem** tailored primarily for students.

---

# 🎯 Problem Statement

Modern campuses face several transportation challenges:

- Traffic congestion  
- Limited parking space  
- Lack of integrated commuting systems  
- Inefficient cycle rental processes  
- Security and transparency issues  
- High carbon footprint  

Existing systems are often fragmented and unreliable. There is a need for a **unified, technology-driven transportation platform** that promotes sustainability while ensuring convenience and security.

# 💡 Proposed Solution

SOLACE introduces a centralized mobile application that integrates:

- 🚖 E-Rickshaw Booking System  
- 🚲 Smart E-Cycle Rental System  
- 🔐 Secure Authentication & OTP Verification  
- 🌍 Sustainable Transportation Model  

The system ensures transparency, accountability, and eco-friendly commuting through IoT-enabled smart locks and real-time data management.

# 📱 Core Modules

## 1️⃣ E-Rickshaw Management System

### 👨‍🎓 Student Interface
- Select pickup & drop location
- Automatic distance-based fare calculation
- OTP generation for ride security
- Secure in-app payment
- Ride feedback submission

### 🚖 Driver Interface
- Real-time ride notifications
- Accept/Reject ride requests
- OTP verification before ride start
- Ride completion & payment trigger

### 🔐 Security Feature
- OTP-based ride authentication
- Transparent fare system
- Real-time communication via Firebase

---

## 2️⃣ E-Cycle Management System

### 🔓 Smart Lock Integration
- Unlock cycle using mobile app
- Controlled via ESP32 microcontroller
- Bluetooth / Wi-Fi communication

### ⏱ Time-Based Billing
- Ride duration tracking
- Automatic fare calculation
- Payment enabled only after proper locking

### 🛡 Smart Lock Safety Mechanism
- IR sensor detects lock position
- Ride cannot be completed unless securely locked
- Prevents misuse and theft
